About CertiFy

CertiFy is a sample frontend for storing and verifying certificates using IPFS and anchoring to blockchain. This UI demonstrates:

  • IPFS file upload flow (client-side)
  • Smart contract call stubs to store CIDs/hashes
  • Verification UI to retrieve metadata and show status

Note: This repo is frontend-only. For production you need a backend or smart contracts + node provider (Infura/Alchemy) and IPFS node or IPFS HTTP API provider.
